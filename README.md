# 🎉 BirthdayVerse

A modern and responsive birthday celebration website built with **React**, **Vite**, and **Tailwind CSS**. It features beautiful animations, personalized birthday wishes, and an elegant user experience.

## 🚀 Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript

## 📦 Installation

```bash
npm install
npm run dev
```

## 🏗️ Build

```bash
npm run build
```
